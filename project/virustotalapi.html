About the Project

This is an EXE and an Installer to be able to right click a file and use your API key to scan the file.

Project Details

  • Language: Python
  • Functionality: Submit files to VirusTotal by right clicking and selecting "Scan with VirusTotal", and retrieve security analysis reports.

This project took about 2 weeks to research and complete. It is a python-based command line interface executable that will scan your file using your VirusTotal API key. The installer is made so the registry edits can be done safely without making the user dig into their files. You can get your own API key at VirusTotal's website
