About the Project

Generates a QR code for the current website. Firefox and Chrome only.

Project Details

  • Languages: Javascript, CSS, and HTML
  • Functionality: Pops up a small window at the top right of your browser, showing a QR code for the current website.

This extension creates a QR code for whatever website you are currently on. It also allows you to change the QR code to scan to whatever you'd like, and lets you change the size and colors.
