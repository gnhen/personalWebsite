About the Project

Python program that switches between two volumes when a specified key is pressed.

Project Details

  • Language: Python
  • Functionality: Allows the user to switch between two specified volumes when a key specified by the user is pressed.

I created this program for a neccessity that I had. I would regularly complete homework in a school provided study room while having my music or tv show on the tv via HDMI. The study room's audio system would periodically raise and lower the volume randomly. I found this annoying, so I created a program that would let me switch to the appropriate volume for the new study room speaker volume.
