About the Project

This is a program to take a screenshot of your screen. Works great for when the subject of the screenshot blurs itself with printscreen.

Project Details

  • Languages: Python & Batch
  • Functionality: Runs a listener that will take a screenshot whenever the \ (backslash) key is pressed.

I started this project so I could learn how to build around the built-in windows functions. This program automatically finds the downloads folder for whatever computer it is held on. Additionaly, the batch file allows for a shortcut to be made of it, which you can put in the Windows Start Menu folder, allowing for a user to search for this program with the Windows key.
