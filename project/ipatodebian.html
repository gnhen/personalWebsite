About the Project

Simple Python & Java converter for taking .ipa files and making tvOS & iOS compatible .deb files.

Project Details

  • Languages: Python & Java
  • Functionality: Python & Java program that extracts an IPA file, packs it into a Debian file, and sets it up for either iOS or tvOS
